## 🚀 How to Run

Make sure you have **Java JDK** installed. You can verify with:

```bash
java -version
```

**Compile and run any file:**

```bash
# Navigate to the topic folder
cd arrays/

# Compile
javac FileName.java

# Run
java FileName
```

**Example:**

```bash
cd linked-lists/
javac SinglyLinkedList.java
java SinglyLinkedList
```
